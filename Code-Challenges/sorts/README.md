## Overview
A Challenge that implements an InsertSort method to sort arrays
based off this Psuedo Code
``` 
1.		InsertionSort(int[] arr)
2.			FOR i = 1 to arr.length
3.				int j <-- i - 1
4.				int temp <-- arr[i]
5.
6.				WHILE j >= 0 AND temp < arr[j]
7.					arr[j + 1] <-- arr[j]
8.					j <-- j - 1
9.
10.				arr[j + 1] <-- temp ```
