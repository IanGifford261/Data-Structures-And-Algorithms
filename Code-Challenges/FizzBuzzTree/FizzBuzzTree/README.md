# LinkedList

**Author**: Ian Gifford
**Version**: 1.0.0

## Overview
A Challenge that takes the values of each node and checks to see if they are divisible by 3, 5, 3 and 5 to replace them with "Fizz", "Buzz", or "FizzBuzz" respectively.

- Divisible by 3 = "Fizz"
- Divisible by 5 = "Buzz"
- Divisibly by 3 and 5 = "FizzBuzz"
